# Decentralized Crowdfundin

decentralized crowdfunding smart contract built with **Foundry**.<br>
This project demonstrates enterprise-grade smart contract development practices, migrating Web2 architectural discipline (TDD, CI/CD, layered structure) into the Web3 space.

---

## Tech Stack & Tools

- **Smart Contract**: Solidity `^0.8.20`
- **Development Framework**: Foundry (Forge, Cast)
- **CI/CD**: GitHub Actions
- **Libraries Used**: OpenZeppelin Contracts (Initializable, ReentrancyGuard where applicable)
